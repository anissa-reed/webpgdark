My Portfolio

Rogue Pickings Website
HTML & CSS Website from Photoshop Design Comp

Rogue Pickings demostrates my ability to create a static website from a Photoshop design comp. This static website introduces the use of a standard text editor, Visual Studio Code and exhibits my progressing capabilities with HTML and CSS. Also, Rogue Pickings website emphasizes the development of clean code in the HTML and CSS documents. Finally, the sample website helps familiarize me with the industry standard of code validation and the necessary early stages of launching a full-scale website.

Jubliee Austen Project
HTML 5 & CSS Website

The Jubilee Austen Project ensures my code foundation is secure by creating a minimally responsive full website from scratch. Specifically, the site starts from a Photoshop design comp, moves to buidling clean code in HTML, and to the final step of styling with CSS. The final website displays a touch of personal customization when compared to Rogue Pickings website by adding fonts and colors that differ from the original design comp. In addition, the website gives a quick overview of my tech journey into the world of design and coding. Currently, the site is a single page but was designed/built with multiple pages in mind.

Holstee Manifesto Exercise

My redesign of the Holstee Mansifesto showcases a portion of my blossoming HTML/5 and CSS skills in a closed text editor. About Me

I have a passion for building a web that is accessible for everyone.

Hi, I'm Anissa, a Web Designer and Developer with a love of clean, elegant design and code. My web design and development career epiphany came after reflecting on my sons' happy faces excited over what they were learning. So, I asked myself, "What would I do if I could do anything I wanted and it gave me that same excitement?" I knew my best skills like patience, dedication, and curiosity in combination with my naturally supportive and empathtic capabililties would shine as a UX Designer and Developer with an eye for accessbility.

Can't wait to work with you!
